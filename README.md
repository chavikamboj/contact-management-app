# 📇 Contact Management App (MERN)

A simple and modern Contact Management Web Application built using the MERN stack.  
Users can add, view, search, and delete contacts easily.

---

## 🚀 Features

- ➕ Add new contact (Name, Email, Phone)
- 📋 View all contacts
- 🔍 Search contacts
- ❌ Delete contact
- 🎨 Clean and modern UI (responsive design)
- 🌐 Full MERN stack integration

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript (React)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **Deployment**: Render (Backend), Vercel (Frontend)
